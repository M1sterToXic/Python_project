# Симуляция дифракции Фраунгофера на дифракционной решётке

## Что такое дифракция Фраунгофера

Дифракция Фраунгофера (или дифракция в параллельных лучах) — это частный случай дифракции, при котором источник света и точка наблюдения находятся на бесконечно большом расстоянии от препятствия. На практике это означает, что на препятствие падает плоский волновой фронт, и наблюдение ведётся в фокальной плоскости линзы или на достаточно большом удалении.

## Дифракционная решётка

Дифракционная решётка — это оптический элемент, содержащий большое количество параллельных щелей (штрихов), расположенных на одинаковом расстоянии друг от друга. Период решётки d — это расстояние между соседними щелями.

### Принцип работы

При падении монохроматического света на дифракционную решётку каждая щель становится вторичным источником когерентных волн. Эти волны интерферируют между собой, создавая дифракционную картину — чередующиеся максимумы и минимумы интенсивности.

### Для чего нужна дифракционная решётка

1. Спектральный анализ — разложение света в спектр для определения его波长ного состава
2. Определение длины волны света
3. В спектрометрах и спектроскопах
4. В лазерной технике для управления пучками

## Основные формулы

### Условие главных максимумов

d * sin(θ) = m * λ

где:
- d — период решётки (м)
- θ — угол дифракции
- m — порядок дифракции (m = 0, ±1, ±2, ...)
- λ — длина волны света (м)

### Положение максимумов на экране

x = L * tan(θ)

где:
- x — расстояние от центрального максимума до максимума порядка m (см)
- L — расстояние от решётки до экрана (м)

### Дисперсия решётки

D = dθ/dλ = m / (d * cos(θ))

### Разрешающая способность

R = λ / Δλ = m * N

где N — общее количество штрихов решётки

## Как работает моя симуляция

Симуляция реализует визуализацию дифракционной картины Фраунгофера с возможностью интерактивного изменения параметров.

### Основные возможности

1. Выбор типа света: белый свет или цветные светофильтры (красный, жёлтый, зелёный, синий, фиолетовый)

2. Регулировка параметров:
   - L — расстояние от решётки до экрана
   - d — период дифракционной решётки

3. Масштабирование изображения для детального рассмотрения

4. Сохранение полученной картины в файл

### Алгоритм визуализации

1. Для белого света симуляция перебирает длины волн в диапазоне 380-760 нм и отображает каждую волну своим цветом

2. Для монохроматического света используется пиковая длина волны, характерная для выбранного фильтра

3. Для каждого порядка дифракции m = ±1, ±2 вычисляется положение полосы на экране

4. Полосы отображаются в виде цветных прямоугольников на чёрном фоне

5. Центральный максимум (m = 0) всегда отображается в центре

### Техническая реализация

- Язык программирования: Python
- Графическая библиотека: Matplotlib
- Интерфейс: PyQt5
- Бэкенд: Qt5Agg

### Управление программой

1. Изменение параметров через поля ввода автоматически обновляет картину через 1.5 секунды после последнего изменения

2. Кнопки масштаба позволяют приближать/отдалять изображение

3. Кнопка "Сохранить картину" сохраняет текущее изображение в файл diffraction_pattern.png

### Ограничения и допущения

1. В симуляции не учитывается изменение интенсивности максимумов с порядком дифракции

2. Ширина дифракционных полос задаётся эмпирически для наглядности

3. Для белого света используется упрощённая модель с равномерным распределением длин волн
