# My Project

## Описание

Проект представляет собой 2D игру, написанную на Python с использованием библиотеки Pygame. Игрок управляет персонажем, который автоматически бежит по лесу. Цель игры — как можно дольше избегать столкновений с врагами.

## Управление

- Стрелка ВЛЕВО — перемещение персонажа назад.
- Стрелка ВПРАВО — перемещение персонажа вперёд.
- ПРОБЕЛ — прыжок. Поддерживается двойной прыжок.

## Механика игры

- Персонаж постоянно бежит вперёд. Игрок может ускорять или замедлять его с помощью стрелок.
- На земле появляются враги-рептилии. Они двигаются справа налево.
- В воздухе появляются враги-орлы. Они также двигаются справа налево, но на большей высоте.
- Чтобы избежать рептилию, игрок должен перепрыгнуть её.
- Чтобы избежать орла, игрок должен либо увернуться, либо перепрыгнуть его в правильный момент, так как орел находится в воздухе.
- При столкновении с любым врагом игра останавливается и появляется экран проигрыша.

## Экран проигрыша

- Появляется затемнённый фон с надписью You luss!
- Появляется кнопка Try again? для перезапуска игры.
- При нажатии на кнопку игра начинается заново: позиция игрока сбрасывается, все враги удаляются, таймеры появления врагов запускаются с новыми случайными интервалами.

## Технические особенности

- Анимация персонажа и врагов меняется с определённой задержкой, не привязанной к частоте кадров.
- Враги появляются не по расписанию, а со случайными интервалами от 3 до 8 секунд для рептилий и от 4 до 9 секунд для орлов.
- Игрок может совершить два прыжка подряд, прежде чем коснётся земли.
- Фоновый лес движется, создавая эффект бесконечного бега.
- Фоновая музыка (лес) воспроизводится в начале игры.

## Запуск игры

Для запуска игры необходимо установить библиотеку Pygame:

pip install pygame

После этого можно запустить файл runner.py.

## Структура проекта

- runner.py — основной файл игры.
- Папка Изображения — содержит спрайты персонажа, врагов и фоны.
- Папка Audio — содержит звуковые файлы.
- Папка Шрифт — содержит шрифт для текста в игре.
